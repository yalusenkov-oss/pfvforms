# 📋 Инструкция по установке кода в Google Apps Script

## Шаг 1: Откройте Google Apps Script

1. Откройте вашу Google Таблицу
2. Нажмите **Расширения** → **Apps Script**
3. Откроется редактор кода

## Шаг 2: Скопируйте код

1. Откройте файл `apps-script-code.js` в этом проекте
2. **Выделите весь код** (Ctrl+A или Cmd+A)
3. **Скопируйте** (Ctrl+C или Cmd+C)

## Шаг 3: Вставьте код в редактор

1. В редакторе Google Apps Script **удалите весь старый код**
2. **Вставьте** скопированный код (Ctrl+V или Cmd+V)
3. **Сохраните** (Ctrl+S или Cmd+S, или кнопка 💾)

## Шаг 4: Проверьте настройки

Убедитесь, что в начале файла указаны правильные названия листов:

```javascript
const DISTRIBUTION_SHEET_NAME = 'Дистрибуция';
const PROMO_SHEET_NAME = 'Промо';
```

**Важно:** Названия должны точно совпадать с названиями листов в вашей таблице!

## Шаг 5: Создайте развертывание

1. Нажмите **Развернуть** → **Новое развёртывание**
2. Нажмите на шестерёнку ⚙️ → **Веб-приложение**
3. Заполните настройки:
   - **Описание**: PFVMUSIC Forms API
   - **Выполнять как**: Я (ваш email)
   - **Доступ**: **Все** (важно!)
4. Нажмите **Развернуть**
5. **Разрешите доступ** (нажмите "Разрешить" во всех окнах)
6. **Скопируйте URL** веб-приложения

## Шаг 6: Обновите URL в проекте

Если URL изменился, обновите его в:

1. **`.env`** файл:
   ```
   GOOGLE_SCRIPT_URL="ваш_новый_URL"
   ```

2. **`test-google-sheets.html`** (если используете):
   - Найдите строку с `const scriptUrl = ...`
   - Замените на новый URL

## Шаг 7: Проверьте работу

1. Откройте `test-google-sheets.html` в браузере
2. Нажмите "Проверить доступность URL" - должно показать ✅
3. Попробуйте отправить тестовые данные
4. Проверьте Google Таблицу - данные должны появиться

## Устранение проблем

### Данные не сохраняются

1. **Проверьте логи в Apps Script:**
   - **Выполнение** → **Журнал выполнений**
   - Посмотрите, какие ошибки возникают

2. **Проверьте названия листов:**
   - Убедитесь, что листы называются точно **"Дистрибуция"** и **"Промо"**
   - Регистр важен!

3. **Проверьте права доступа:**
   - При развертывании должно быть выбрано "Все"
   - Создайте новое развертывание если нужно

### Ошибка "Sheet not found"

- Проверьте, что листы существуют в таблице
- Проверьте названия листов (точное совпадение!)

### Ошибка CORS

- Убедитесь, что развертывание настроено как "Веб-приложение"
- Проверьте, что доступ установлен на "Все"
