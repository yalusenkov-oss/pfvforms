🌐 React App
Клиентский сайт. SPA на React, задеплоен на Vercel. Два маршрута: /# (форма) и /#sign?token=... (подписание договора).
⚡ Vercel API
Serverless-функции: /api/submit — приём формы и отправка email; /api/_email — HTML шаблон письма; /api/test-email — диагностика.
📊 Google Apps Script
Основной бэкенд: сохраняет данные в Google Sheets, генерирует HTML договора, создаёт токены подписания, сохраняет PDF в Google Drive.
🔐 Хранилище
Google Sheets — база данных (дистрибуция, промо, промокоды). Google Drive — PDF договоров и обложки.

Vercel вызывает GAS action update_contract_tg_status. GAS находит message_id и редактирует через editMessageText.

То же сообщение, финальное редактирование ✏️

▸ Лицензиар: Имя артиста

▸ Произведение: Название трека
📄 Открыть подписанный договор
При подписании GAS читает tg_contract_msg_id и вызывает editMessageText. Fallback: новое сообщение если редактирование не удалось.

Fallback при ошибке
Если Yandex SMTP отклонил письмо (spam 554) — Vercel вызывает GAS update_contract_tg_status, и в Telegram редактируется сообщение с пометкой "⚠️ Email не доставлен".
HTML vs Plain Text
Оба варианта: HTML для email-клиентов и plain-text fallback. Все inline-стили, адаптив для мобильных через media query + box-sizing:border-box на кнопке.

10. Google Apps Script — все actions (doPost)
distribution
saveDistribution()
Сохраняет заявку, генерирует договор, создаёт sign token, отправляет TG-уведомление.
sign_get
getSignData()
Возвращает HTML договора по токену. Проверяет срок действия токена.
sign_submit
submitSignature()
Вшивает подпись в HTML, создаёт PDF, загружает на Drive, отправляет TG.
sign_create
createSignLink()
Создаёт новый токен подписания (из админки). Отправляет TG.
promo
savePromo()
Сохраняет промо-заявку в отдельный лист. Telegram #питчинг.
promo_code
savePromoCode()
Создаёт промокод. Telegram #промокод. Только из админки.
update_contract_tg_status
updateContractTgStatus()
Редактирует TG-сообщение по contractNumber. Добавляет статус email.
get_promo_codes (GET)
doGet()
Возвращает список активных промокодов для клиентской формы.
delete / promo_code_delete
deleteRowData() / deletePromoCode()
Удаление записей. Только из админки.
generate_contract
generateContract()
Ручная генерация договора из админки для существующей записи.
